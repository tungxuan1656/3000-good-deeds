# 08 - Manual Work Checklist

## Việc manual bắt buộc trước khi go-live

- Chọn domain chính thức cho landing page.
- Cấu hình DNS về Vercel.
- Xác minh domain trong Google Search Console.
- Tạo và upload ảnh thật thay toàn bộ placeholder.
- Viết lại alt text thực tế theo ảnh cuối cùng.
- Tạo ảnh OG chính thức (`1200x630`).
- Kiểm tra chính tả tiếng Việt toàn bộ nội dung.
- Kiểm tra link CTA mở đúng webapp chính.
- Kiểm tra hiển thị trên iPhone/Android thực tế.

## Việc manual sau go-live

- Submit sitemap trong Search Console.
- Request indexing URL homepage.
- Theo dõi lỗi Coverage/Pages trong 7-14 ngày đầu.
- Theo dõi Core Web Vitals trong Search Console.
- Tối ưu title/meta nếu CTR thấp.

## Việc manual định kỳ

- Cập nhật screenshot khi UI webapp chính đổi lớn.
- Cập nhật FAQ theo câu hỏi người dùng thật.
- Rà soát broken links mỗi tháng.
- Đánh giá lại keyword target mỗi quý.
