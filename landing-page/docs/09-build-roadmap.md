# 09 - Build Roadmap

## Tổng quan lộ trình

1. Bước 1 - Tạo bộ tài liệu dự án
2. Bước 2 - Scaffold dự án Next.js trong `landing-page`
3. Bước 3 - Dựng UI landing page + placeholder ảnh
4. Bước 4 - Tích hợp SEO kỹ thuật đầy đủ
5. Bước 5 - QA responsive + Lighthouse + accessibility
6. Bước 6 - Deploy Vercel + cấu hình domain + submit Search Console
7. Bước 7 - Theo dõi chỉ số SEO và tối ưu vòng lặp

## Chi tiết từng bước

### Bước 1 - Tạo bộ tài liệu dự án (Hoàn thành)

- Chốt mục tiêu sản phẩm.
- Chốt định hướng UI/UX.
- Chốt checklist SEO bắt buộc.
- Chốt danh sách ảnh placeholder.
- Chốt kế hoạch Next.js + Vercel.
- Chốt hướng dẫn kiểm tra SEO và việc manual.

### Bước 2 - Scaffold Next.js

- Khởi tạo app Next.js + TypeScript + Tailwind.
- Thiết lập cấu trúc thư mục theo kế hoạch.

### Bước 3 - Xây dựng giao diện

- Dựng các section theo cấu trúc content.
- Thêm CTA điều hướng sang webapp chính.
- Dùng placeholder cho toàn bộ ảnh chưa có asset thật.

### Bước 4 - SEO kỹ thuật

- Metadata, OG/Twitter card.
- `robots.txt`, `sitemap.xml`, canonical.
- JSON-LD schema.

### Bước 5 - QA

- Lint + build.
- Lighthouse và sửa lỗi quan trọng.
- Test mobile thực tế.

### Bước 6 - Deploy

- Deploy Vercel.
- Cấu hình domain.
- Kết nối Search Console.

### Bước 7 - Tối ưu sau release

- Theo dõi query và CTR.
- Cập nhật content/FAQ.
- Cập nhật ảnh và bằng chứng sử dụng thực tế.
