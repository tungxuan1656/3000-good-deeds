# Landing Page SEO - 3000 Việc Thiện

Thư mục này là điểm khởi đầu cho dự án landing page bằng Next.js, mục tiêu SEO để dẫn traffic về web chính: `https://3000-viec-thien.web.app/`.

## Mục tiêu

- Xây landing page tiếng Việt, giới thiệu ý nghĩa dự án thiện nguyện.
- Làm rõ thông điệp: website miễn phí vĩnh viễn.
- Tối ưu SEO kỹ thuật để có thể index và hiển thị tốt trên Google Search.
- Có các CTA rõ ràng để mở webapp chính.
- Triển khai chuẩn Vercel.

## Trạng thái hiện tại

- `Bước 1` đã hoàn thành: tạo bộ tài liệu nền cho toàn dự án.
- `Bước 2` đã hoàn thành task đầu tiên: scaffold Next.js + TypeScript + Tailwind trong `landing-page`.
