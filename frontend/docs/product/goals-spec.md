# Goals and Goal History Spec

## Goal

Help users set direction through weekly/monthly/yearly kindness targets.

## Goal Types

- `weekly`
- `monthly`
- `yearly`

## Core Rules

- max one goal per type per user
- no punitive language for unmet targets
- goal history is reflective, not judgmental
- historical records should not be retroactively fabricated

## Main Flows

- enable/update/disable goals
- track current period progress
- browse historical periods

## Edge Cases

- period transitions
- target updates mid-period
- deed updates/deletions impacting counters
