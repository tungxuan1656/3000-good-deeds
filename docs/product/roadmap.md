# Product Roadmap

## Phase 1: Core Foundation

- infrastructure setup (monorepo, DB, backend, frontend)
- authentication (Firebase identity + backend session)
- profile/settings basics
- kindness log CRUD

## Phase 2: Reflection

- weekly/monthly statistics
- category distribution
- streak tracking

## Phase 2.5: Inner Cultivation

- random wisdom quotes
- random acts of kindness
- private soul journal
- meditation and breathing support

## Phase 3: Habit Support

- goals (weekly/monthly/yearly)
- reminders and notifications

## Phase 4: Stability and Growth

- UX polish
- PWA readiness
- multilingual support

## Implementation Principles

- mobile-first
- simple by default
- privacy-first
