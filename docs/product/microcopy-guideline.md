# Microcopy Guideline

## Principles

- compassionate and non-judgmental tone
- no guilt-triggering language
- no gamified language (score/rank/karma points)
- privacy-first wording
- realistic and honest messaging

## Voice and Tone

- short, calm, supportive
- suggest one next action at a time
- avoid commanding or moralizing tone

## Preferred Terms

- Kind deed
- Reflection
- Gentle reminder
- Body / Speech / Mind

## Avoid Terms

- karma score, rank, leaderboard
- fail, punishment, guilt phrasing

## CTA Style

Prefer:
- Save
- Record
- Try now
- Maybe later

Sensitive actions must be explicit:
- Delete this entry
- Delete account
- Disable reminder

## State Messaging

Define consistent copy for:
- empty states
- success feedback
- confirmation dialogs
- network and unknown errors
- permission prompts
