# Ethics and Product Principles

## Privacy First

- user data belongs to users
- private reflections are private by default
- no ad-targeting data exploitation

## No Dark Patterns

- no spam notifications
- no guilt-based messaging
- account deletion must be straightforward

## Truthfulness

- no exaggerated claims
- no pseudo-scientific promises
- transparent behavior in randomization and system logic

## Humane Design

- calm visual tone
- compassionate language
- accessibility-conscious interfaces
