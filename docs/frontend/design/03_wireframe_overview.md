# D03 – Wireframe Overview

## Nguyên tắc wireframe
- Mobile-first
- Ít thành phần trên mỗi màn hình
- Ưu tiên hành động ghi nhận việc thiện

## 1. Home Screen

**Bố cục:**
- Header: ngày hôm nay + câu nhắc
- Trung tâm: nút lớn "+ Việc thiện"
- Dưới: việc thiện gần nhất

```
[ Câu nhắc chánh niệm ]

[  + Ghi nhận việc thiện  ]

Việc gần nhất:
- Giúp người qua đường
```

## 2. Ghi nhận việc thiện

```
Chọn danh mục:
[ Lời nói ] [ Hành động ]
[ Ý nghĩ ] [ Khác ]

Mô tả (tuỳ chọn)
[ ............. ]

Cảm xúc (tuỳ chọn)
[ Bình an ▾ ]

[ Lưu lại ]
```

## 3. Thống kê

```
Tổng: 245 / 3000

[ Biểu đồ cột theo tuần ]

Danh mục nhiều nhất:
- Gia đình
```

## 4. Hồ sơ cá nhân

```
Tên / Pháp danh

Mục tiêu:
3000 việc thiện

[ Thành tựu ]
[ Cài đặt ]
```

## Ghi chú
- Wireframe chỉ định hướng, không phải UI cuối
- Thiết kế chi tiết sẽ nằm trong tài liệu UI Style
