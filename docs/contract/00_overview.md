# CONTRACT – OVERVIEW (FE ↔ BE)

Tài liệu giúp frontend và backend có cùng ngôn ngữ về **tính năng, dữ liệu, API**.

## Thứ tự đọc gợi ý
1. `contract/features/*` (đặc tả tính năng)
2. `contract/data/*` (nguyên tắc dữ liệu)
3. `contract/database_feature_mapping.md` (đối chiếu dữ liệu ↔ tính năng)
4. `contract/api_overview.md` (danh sách endpoint)
5. `contract/api/*` (request/response chi tiết)
