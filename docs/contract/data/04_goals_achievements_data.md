# F-DATA · Goals & Achievements

## 1. Goals (Mục tiêu)

### 1.1. Dữ liệu mục tiêu

| Trường       | Mô tả              |
| ------------ | ------------------ |
| goal_id      | ID mục tiêu        |
| user_id      | Người đặt          |
| title        | Tên mục tiêu       |
| target_value | Số lượng mong muốn |
| duration     | Thời gian          |
| created_at   | Ngày tạo           |

### 1.2. Ví dụ
- “Mỗi ngày 1 việc thiện trong 30 ngày”
- “Giữ lời nói chánh niệm trong 1 tuần”

---

## 2. Achievements (Thành tựu)

| Trường         | Mô tả         |
| -------------- | ------------- |
| achievement_id | ID            |
| title          | Tên thành tựu |
| description    | Ý nghĩa       |
| unlocked_at    | Thời điểm đạt |

---

## 3. Triết lý thiết kế

- Thành tựu mang tính **khích lệ nhẹ**
- Không dùng từ ngữ phán xét
- Không so sánh giữa người với người
