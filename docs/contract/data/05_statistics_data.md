# F-DATA · Statistics Data

## 1. Mục đích

Thống kê nhằm:
- Giúp người dùng tự soi lại hành trình
- Nhận ra sự chuyển hóa theo thời gian

---

## 2. Dữ liệu thống kê chính

| Chỉ số            | Mô tả                |
| ----------------- | -------------------- |
| total_deeds       | Tổng việc thiện      |
| streak_days       | Chuỗi ngày liên tiếp |
| deeds_by_category | Theo nhóm            |
| monthly_summary   | Tổng kết tháng       |

---

## 3. Nguyên tắc hiển thị

- Dữ liệu chỉ phục vụ **chính người dùng**
- Không hiển thị như KPI hay thành tích
- Dùng ngôn ngữ nhẹ nhàng: “Bạn đã đi được …”
