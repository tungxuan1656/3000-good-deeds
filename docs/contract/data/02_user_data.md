# F-DATA · User Data

## 1. Mục đích

User Data lưu trữ các thông tin cần thiết để:
- Cá nhân hóa trải nghiệm tu tập
- Theo dõi tiến trình làm việc thiện
- Không xâm phạm đời sống riêng tư

---

## 2. Thông tin người dùng

### 2.1. Thông tin cơ bản

| Trường       | Mô tả                   |
| ------------ | ----------------------- |
| user_id      | ID duy nhất             |
| display_name | Tên hiển thị            |
| avatar       | Ảnh đại diện (tuỳ chọn) |
| created_at   | Ngày tạo tài khoản      |

### 2.2. Thông tin tùy chọn

| Trường         | Mô tả                              |
| -------------- | ---------------------------------- |
| age_range      | Khoảng tuổi (không bắt buộc)       |
| location       | Quốc gia / vùng                    |
| buddhism_level | Người mới / đã thực hành / lâu năm |

---

## 3. Cài đặt cá nhân

| Thuộc tính      | Mô tả                      |
| --------------- | -------------------------- |
| reminder_time   | Giờ nhắc làm việc thiện    |
| notification_on | Bật/tắt thông báo          |
| theme           | Giao diện sáng / tối       |
| privacy_mode    | Riêng tư / chia sẻ hạn chế |

---

## 4. Nguyên tắc lưu User Data

- Không yêu cầu thông tin nhạy cảm
- Không lưu tôn giáo bắt buộc
- Người dùng có thể xoá dữ liệu bất kỳ lúc nào
