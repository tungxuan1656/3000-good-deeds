# F-DATA · Good Deed Data

## 1. Khái niệm

**Việc thiện** là mọi hành động:
- Có lợi cho người khác
- Giúp thân – khẩu – ý thanh tịnh
- Không cần lớn lao

---

## 2. Cấu trúc dữ liệu việc thiện

| Trường        | Mô tả                |
| ------------- | -------------------- |
| deed_id       | ID việc thiện        |
| user_id       | Người thực hiện      |
| category_id   | Nhóm việc thiện      |
| description   | Mô tả ngắn           |
| deed_date     | Ngày thực hiện       |
| emotion_after | Cảm nhận sau khi làm |
| is_private    | Riêng tư / chia sẻ   |
| created_at    | Thời điểm ghi nhận   |

---

## 3. Phân loại việc thiện

- Giúp người
- Giúp vật
- Giữ giới
- Tu tập nội tâm
- Lời nói lành
- Ý nghĩ thiện

---

## 4. Nguyên tắc ghi nhận

- Người dùng **tự nguyện ghi nhận**
- Không đánh giá đúng/sai
- Không có điểm thưởng mang tính ganh đua
