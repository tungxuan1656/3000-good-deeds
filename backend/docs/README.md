# Backend Docs

Backend documentation is organized into three groups:

- `backend/docs/product/`: backend business behavior and domain rules.
- `backend/docs/technical/`: API, DB, auth, and implementation details.
- `backend/docs/standards/`: mandatory coding and review standards.

## Suggested Reading Path

1. `backend/AGENTS.md`
2. `backend/docs/standards/README.md`
3. `backend/docs/technical/api-design.md`
4. `backend/docs/technical/api-reference.md`
5. `backend/docs/technical/database-schema.md`
6. `docs/monorepo/architecture.md`
