## Project Overview
A platform to encourage, record, and celebrate good deeds. Users can log activities, set goals, join categories, and track their progress. The project aims to foster a positive community and personal development.

## Vision & Philosophy
- **Vision:** Empower individuals to make a positive difference, one good deed at a time.
- **Philosophy:** Ethical, privacy-first, and user-centric design. See `docs/vision/` for more.

## Architecture
- **Backend:** TypeScript, Cloudflare Workers, SQL migrations, RESTful API.
- **Frontend:** React, Vite, PWA-ready, modern UI/UX.
- **Docs:** Comprehensive documentation in `docs/`.

See `docs/backend/02_architecture.md` and `docs/frontend/design/04_frontend_architecture.md` for details.
